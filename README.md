# 报价单结算管理系统
一款报价单管理工具，用于报价信息录入、自动计算和清单打印。

##  核心说明
###  支持录入的核心字段
| 字段名称       | 说明                 | 示例               |
|----------------|----------------------|--------------------|
| 工作内容       | 具体服务/施工内容    | 墙面刷漆、电路改造 |
| 材料           | 使用的主材/辅材      | 乳胶漆、铜线       |
| 规格/尺寸      | 材料规格/施工尺寸    | 5L/桶、3m×2.8m     |
| 数量           | 施工/材料数量        | 2桶、10㎡          |
| 价格           | 单价（元）           | 180、80            |
| 总价           | 自动计算（数量×价格）| 360、800           |
| 经办人         | 对接人/负责人        | 张三               |
| 备注           | 特殊说明             | 不含高空作业费     |

###  核心操作功能
- ✅ 信息录入：填写上述字段，自动计算总价，避免手工算错
- ✅ 数据管理：对报价记录进行新增、编辑、删除
- ✅ 清单打印：一键生成格式化结算清单，支持打印/导出
- ✅ 本地存储：数据保存在浏览器，刷新/重启不丢失

##  快速使用
###  环境准备
- 现代浏览器（Chrome/Firefox/Edge）

###  操作步骤
1. 启动项目后，在表单中填写「工作内容、材料、规格、数量、价格」等信息；
2. 系统自动计算「总价」，补充「经办人、备注」后点击「保存记录」；
3. 所有记录会显示在下方表格中，可编辑/删除任意记录；
4. 确认所有记录无误后，点击「打印结算清单」即可生成可打印的报价单。
